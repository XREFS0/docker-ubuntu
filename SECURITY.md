## Reporting a Vulnerability

We take the security of our project very seriously. If you discover a vulnerability, we would like to know about it so we can take steps to address it as quickly as possible.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via our official contact channels (such as Telegram or Facebook) listed in our `README.md`.

When reporting, please provide:
* A description of the vulnerability.
* Steps to reproduce the issue.
* Any possible solutions you might have.

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress.
